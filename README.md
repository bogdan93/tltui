# Time Logger TUI

A terminal-based time tracking application built with Go and Bubble Tea.

## Installation

### Prerequisites
- Go 1.16 or higher
- SQLite3

### Linux

```bash

# Clone the repository
git clone <repository-url>
cd tltui

# Build and install
cd src
go build -o tltui
sudo mv tltui /usr/local/bin/

# Run
tltui
```

### macOS

```bash
# Install Go if not already installed
brew install go

# Clone the repository
git clone <repository-url>
cd tltui

# Build and install
cd src
go build -o tltui
mv tltui /usr/local/bin/

# Run
tltui
```
