# Time Logger TUI

A terminal-based time tracking application built with Go and Bubble Tea.

## Installation

### Prerequisites
- Go 1.25.1 or higher

### Linux

```bash

# Clone the repository
git clone https://github.com/bogdan93/tltui.git
cd tltui

# Build && Install
make build
sudo mv bin/app /usr/local/bin/

# Run
tltui
```
